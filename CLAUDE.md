# 📄 엑셀 출력 최적화 프로그램 개발 요청 (Claude)

## 1. 배경 및 현황

* **업무 환경:** 외부 인터넷 접속이 불가능한 **폐쇄망 환경**에서 클라우드 콘솔 사용자 요청 처리(MSP) 업무 수행.
* **문제점:** 사용자 요청 엑셀 파일에 폐쇄망 작업 공간에서 접근 불가.
* **현행 작업:** 다양한 서식의 엑셀 파일을 **A4/A3 용지에 수동으로 레이아웃을 정리 및 최적화하여 출력** 후 작업에 활용.
* **비효율:** 다수 컬럼과 복잡한 서식으로 인해 수동 정리 및 출력 과정에 비효율 발생.

---

## 2. 목적

* **엑셀 데이터를 A4 또는 A3 용지 규격에 맞춰 자동으로 정리 및 최적화하여 출력할 수 있는 프로그램 개발.**

---

## 3. 개발 환경

* **언어:** Python
* **실행 환경:** Windows OS
* **GUI 라이브러리:** PySide6

---

## 4. 핵심 기능 요구사항

### 4.1. 데이터 입/출력 기능

1.  **엑셀 파일 불러오기:** 엑셀(XLSX/XLS) 파일 로드 기능.
2.  **클립보드 붙여넣기:** **엑셀에서 복사한 내용**을 프로그램 내 그리드(Grid)에 붙여넣기.
3.  **출력/저장 기능:** 최종적으로 정리 및 최적화된 데이터를 **엑셀 파일로 저장**하는 기능.

### 4.2. 설정 및 메뉴

* **상단 메뉴**를 통해 다음 항목 설정 제공:
    * **용지 선택:** A4 (기본값), A3
    * **방향 선택:** 가로 (기본값), 세로
    * **글자 크기 조절:** 숫자 입력 필드 (기본값: 10pt)

### 4.3. 그리드 데이터 자동 정렬 및 최적화 로직

| 기능 | 상세 로직 |
| :--- | :--- |
| **폰트/크기 변환** | 그리드 데이터는 **상단 메뉴에 설정된 글자 크기로 일괄 변환**되어야 함. (글꼴 종류는 기본값 사용) |
| **빈 셀 최적화** | **값이 없는 셀(헤더 제외)**이 있는 컬럼은, 해당 컬럼의 **헤더 글자 크기를 최소 8pt까지 줄이고** **셀의 너비를 최대한으로 줄임.** |
| **컬럼 값 Bold 처리** | **각 컬럼별**로 값을 비교하여, **앞에서부터 텍스트가 같은 공통 영역**을 **Bold 처리**해야 함. |
| **헤더 자동 줄바꿈** | **컬럼 값의 자리수가 짧고 헤더가 긴 경우**, 셀 너비를 줄이는 대신 **헤더는 여러 줄로 표현**되어야 함 (텍스트 래핑 적용). |
