# 새로운 기능 추가

# 기능 설명
- GridWidget 의 셀의 문자열 중 일 부부분 텍스트 색상 변경 기능
- 예를들어 ABC123DEF 라는 문자열이 있을 때 123 만 색상 변경 또는 bold 처리

# 왜 필요한가?
- 데이터를 가공해서 엑셀로 출력한 후에 프린트된 출력물을 보고 폐쇄망 환경에서 수작업 진행
- 표에서 끝 부분만 다른 글자들이 많은데 이런데이터를 손쉽게 파악하기 위해서 임
- 예를 들어 알래와 같은 데이터들이 있음
|col1|col2|col3|
|-|-|-|
|vs-dev-01|sbnet-dev-01|lb-my-dev-01-south1|
|vs-dev-02|sbnet-prd-01|lb-my-dev-02-south1|
|vs-dev-03|sbnet-stg-01|lb-my-dev-03-south1|

# 주의사항
- 그리드의 데이터를 엑셀로 export할 때 일부 문자열 스타일 정보가 엑셀에도 반영되어야 함.

# 최종 결정
- 위에서 왜 필요한가? 항목에서 예시를 들었던 표와 같은 경우 하이브리드 방식으로 구현해서 표에서 행의 값이 다른 부분은 빨강색으로 표시하는게 가능한지? 확인해서 가능하다면 상세 작업 계획서를 별도의 문서로 저장해줘.
