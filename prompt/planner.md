# 배경
나는 클라우드 콘솔에서 사용자의 요청을 처리해주는 msp 업무를 하고 있어.
근무환경은 외부 인터넷 접속이 되지 않는 폐쇄망 환경에서 작업을 해
사용자의 요청사항은 엑셀로 관리되고 있는데 폐쇄망 작업 공간에서는 사용자의 요청 엑셀 파일에 접근할 수 없어.
그래서 다양한 형태의 엑셀 파일을 a4 또는 a3 용지에 출력을 해서 작업공간에서 업무를 수행중이야.
엑셀 문서는 많은 컬럼이 존재하고 글꼴, 폰트 크기, 서식 등 다양하기 때문에 이러한 문서를 출력했을 때
a4 용지에 담아내기가 불충분해. 그래서 수적업으로 a4 용지에 최대한 보기 편하도록 수작업을 한 후에 프리트 작업을 해


# 목적
엑셀 파일을 a4 또는 a3 용지에 맞게 정리되서 출력할 수 있는 프로그램을 만들려고 해.

# 필요한 기능
1. 엑셀 팡일 불러오기 기능
2. 엑셀에서 복사한 내용을 파이썬 그리드에 붙여넣기 기능
3. 상단 메뉴
    - 용지 선택 (a4(default), a3)
    - 가로(default), 세로 선택
    - 글자 크기 조절 ( default : 10)
4. 그리드에 들어온 데이터는 아래와 같이 자동 정렬되어야 해.
    - 글꼴, 크기는 상단 메뉴에 설정된 값을 변환
    - 헤더가 아닌 셀에 값이 없는 경우에는 헤더의 글자 크기를 최소 8까지 줄이고 셀의 너비를 최대한으로 줄이기
    - 각 컬럼 기준으로 앞에서부터 값이 같은 텍스트는 bold 처리해주기
        - 예> A1, A2, A3 값이, abc-1, abc-2, abc-ddd-4 이라고 하면 abc- 영역은 볼드 처리해주기
    - 각 컬럼의 값들이 자리수가 짧고, 컬럼의 헤더가 긴 경우 셀의 너비를 줄이고 헤더는 여러줄로 표현되기
